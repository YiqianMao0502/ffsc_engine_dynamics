# 周闯论文第 2 章（第 61–70 页）实现笔记

## 2.4.3 冷却通道动力学模型
- 冷却通道采用多节点热网络形式（图 31、图 32），节点包括：燃烧室热气体、热障涂层/内壁各层、冷却剂主通道及旁通支路。模型以轴向离散的控制体描述质量、动量与能量交换。
- **节点能量守恒（式 2.75）**：每个节点满足 `m_i c_{p,i} dT_i/dt = \sum \Phi_{i,\text{in}} - \sum \Phi_{i,\text{out}} + \dot m_{i-1} h_{i-1} - \dot m_i h_i`，用于冷却剂、壁面及热障层的瞬态温度更新。
- **换热/导热通量定义（式 2.76–2.78）**：
  - 对于流体与固体之间的对流换热，`\Phi = h A (T_f - T_w)`；
  - 对于圆柱壁的径向导热，`\Phi = 2 \pi \lambda L (T_{r_1} - T_{r_2}) / \ln (r_2 / r_1)`；
  - 对于平板导热，`\Phi = \lambda A (T_{x_1} - T_{x_2}) / b`。
- **复合壁导热叠加（式 2.79–2.81）**：多层壁面通过热阻串联/并联组合计算整体热通量；模型分别给出了环向隔热块、矩形肋片与矩形腔体的热阻表达，可直接实现热阻网络求解。
- 文中强调热障涂层/基材物性随温度的非线性，可在实现中通过温度依赖的 `λ(T)`、`c_p(T)` 函数或查表补齐。

## 2.5 两相流和混合气组件动力学模型
- 第 2.5 节开头（第 70 页）给出了后续章节所用的守恒方程形式：
  - **质量守恒（式 2.82）**：`d(ρV)/dt = \dot m_{in} - \dot m_{out}`；
  - **能量守恒（式 2.83）**：`d(ρ u V)/dt = \dot m_{in} h_{in} - \dot m_{out} h_{out} + \Phi_q`；
  - **混合气能量守恒（式 2.84）**：针对多组分气体，给出了焓/内能与组分质量分数的耦合形式。
- 这些守恒式是 `chapter_2/section_2_5_two_phase_mixed` 目录下 Line/Plenum/Valve 组件代码的理论依据，后续实现 TwoPhaseThermo / GasMixtureThermo 接口时需保证提供 `h(p,T)`, `u(p,T)`, `ρ(p,T)`, `γ_s`, 以及热通量 `Φ_q` 等量。
