## 2.5.2 两相流组件动力学模型

### 管路能量方程（式 2.85–2.88）
- 沿管长坐标 $r$ 的总能量微分写为
  $$\frac{dU}{dr} = \dot{m}\,h + Q$$
  其中 $Q$ 体现轴向导热与壁面热流贡献。
- 对壁面传热采用导热方程 (2.86)：$q_w = -k_w \frac{dT_w}{dr}$，结合壁面热容 $c_w$，得到壁温随时间的演化式 (2.87)。
- 能量守恒与壁面导热耦合，需同时求解流体比焓 $h$、壁面温度 $T_w$ 和局部热流密度。

### 摩擦压降与加速压降（式 2.89–2.92）
- 壁面摩擦压降采用 Churchill 关联：
  $$\frac{dp_f}{dr} = - \frac{G^2}{2\rho D} f_{\text{Churchill}}(Re,\varepsilon/D)$$
- 两相摩擦项按 Lockhart–Martinelli 参数 $X_{tt}$ 与气、液相摩擦系数加权。
- 加速压降由品质数 $x$、气液速度 $v_v, v_l$ 及内部能量 $e$ 计算：
  $$\frac{dp_{acc}}{dr} = -\dot{m}\frac{de}{dr}$$
  需要物性模块提供相平衡密度与比焓导数。

## 2.5.3 混合气体管路及阻性模型

### 流量方程（式 2.95–2.96）
- 质量流量：
  $$\dot{m} = A C_q C_m \frac{p_{up}}{\sqrt{T_{up}}}$$
- 焓流量：
  $$\dot{h} = \dot{m} \cdot h(p_{up}, T_{up})$$
  需要气相物性接口提供 $h$ 和 $T$ 依赖。

### 临界压比与流量系数
- 临界压比 (2.98)：
  $$\left(\frac{p_{cr}}{p_{up}}\right) = \left[\frac{2}{\gamma_s + 1}\right]^{\frac{\gamma_s}{\gamma_s - 1}}$$
- $C_q$ 多项式 (2.97)：$C_q = a_0 + a_1 \eta + a_2 \eta^2 + a_3 \eta^3$，系数由阀门类型决定。
- $C_m$ 分段 (2.100)：
  - 若 $\eta < \eta_{cr}$，则取壅塞流常数
  - 否则按经验式 $C_m = 1 - b_1(1-\eta)^{b_2}$

### 传热换热系数（式 2.101–2.103）
- 雷诺数定义 (2.101)：$Re = \frac{4 \dot{m}}{\pi D \mu}$
- 努塞尔数分段 (2.102)：层流段 $Nu=4.36$，湍流段 $Nu = 0.027 Re^{0.8} Pr^{1/3}$，中间用双曲正切平滑。
- 对流换热系数 (2.103)：$h_c = \frac{Nu \lambda}{D}$，需要混合气输运物性 (黏度 $\mu$、导热系数 $\lambda$) 与 Prandtl 数。
